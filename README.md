# Nassau Candy Analytics Dashboard

## Overview

The Nassau Candy Analytics Dashboard is a Business Intelligence and Data Analytics project developed using Python and Streamlit. The dashboard provides insights into product profitability, revenue performance, margin analysis, regional performance, division-wise contribution, and Pareto Analysis (80/20 Rule).

## Features

* Interactive Dashboard Filters
* Revenue Analysis by Division
* Revenue Analysis by Region
* Product Profitability Analysis
* Top and Bottom Performing Products
* Division Performance Analysis
* Cost vs Profit Analysis
* Pareto Analysis (80/20 Rule)
* Executive Business Insights
* Download Filtered Dataset

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset

Nassau Candy Distributor Sales Dataset

## Project Objective

To analyze profitability, margin performance, division contribution, and product performance for Nassau Candy Distributor using Business Intelligence techniques and interactive visualizations.

## Installation

1. Clone the repository

git clone https://github.com/tosif2027/nassau-candy-analytics-dashboard.git

2. Install dependencies

pip install -r requirements.txt

3. Run the application

streamlit run app.py
